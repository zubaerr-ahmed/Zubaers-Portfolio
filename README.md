# Zubaer's Portfolio

Project 1: Spotify Top 50 Charts Analysis

This is a personal project done by me that analyzes 2023-2024 Spotify Top 50 chart data to uncover trends in popularity, content, and listener behavior across regions, time periods, and song characteristics.

* Datasets were downloaded from Kaggle and a Jupyter notebook was opened to document the data cleaning process
* Loaded required libraries (pandas, numpy, matplotlib & seaborn) on Python and cleaned the dataset to remove missing values, add "Region" and "Duration (minutes)" column
* Merged all the datasets into a master dataset "spotify_merged" to use for data visualization on Tableau
* Created necessary calculated fields to conduct deep-dive analysis
